# Web Template

Vite + Svelte template with good stuffs.

> __-__ Tailwind  
> __-__ pnpm repo  
> __-__ stylelint  
> __-__ prettier disabled!  
> __-__ .vscode config and recommended list  
> __-__ gh-pages  
> __-__ proxy settings

## Usage

Get this repo with `degit`.

```sh
degit https://github.com/rytsh/vitesse.git my-new-project
```

```sh
cd my-new-project
pnpm install
```

Run project

```sh
pnpm run dev
```

Build

```sh
pnpm build
```
